# 🍝 O Rei das Massas - Automação de Pedidos

> **Do cardápio à entrega: transformando a experiência de pedidos de delivery com agilidade e design.**

Este projeto é uma plataforma de automação de pedidos desenvolvida para otimizar a operação de um restaurante de massas. O foco principal foi criar uma jornada do usuário fluida (User Experience), garantindo que a escolha de produtos e a finalização via **WhatsApp** sejam rápidas e intuitivas.

## ✨ Funcionalidades em Destaque

- **🛒 Carrinho Dinâmico:** Gerenciamento de itens em tempo real com atualização automática de valores sem recarregamento de página.
- **➕ Upselling de Extras:** Seção estratégica de ingredientes adicionais para personalizar o pedido e otimizar as vendas.
- **📱 Mobile First:** Layout totalmente responsivo, garantindo uma experiência perfeita em smartphones e tablets.
- **💬 Checkout via WhatsApp:** Integração que envia o pedido formatado diretamente para o número do restaurante, eliminando erros de comunicação.

---

## 🛠️ Tecnologias e Conceitos de Engenharia

- **JavaScript (Vanilla):** Lógica pura para manipulação do DOM e gerenciamento de estado do carrinho.
- **CSS3 Moderno:** Utilização de Flexbox, Grid e animações suaves para feedback visual ao usuário.
- **HTML5 Semântico:** Estrutura organizada visando acessibilidade e SEO.
- **Arquitetura Clean Front-end:** Separação clara entre a estrutura de dados dos pratos e a lógica de renderização.
